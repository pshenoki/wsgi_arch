blind faith - Недостаточная проверка входных данных:
    1. При вводе пользователем русских букв, оттображается неверная кодировка.
        Метод устранения: проверить функции парсинга данных, разобраться с кодировками, возможно дело в html файлах.

    2. Отсутстивие проверки наличия категории для курса(при создании курса в несуществующей категории программа падает).
        Метод устранения: добавить дополнительную проверку.

    3. При регистрации на курс студента отсутствует проверка наличия студента.
        Метод устранения: добавить дополнительную проверку.

    4. При запросе json несуществующей категории программа падает.
        Метод устранения: добавить дополнительную проверку.
